## 💰 FinUp

Um aplicativo desenvolvido com o objetivo de ajudar usuários a organizarem suas finanças de forma simples, prática e eficiente.

## 📌 Sobre o Projeto

Este é um projeto acadêmico desenvolvido em grupo por alunos da faculdade **Unisuam**, com supervisão do professor **Vinicius Pinto**. A proposta do aplicativo é oferecer uma ferramenta acessível para controle financeiro pessoal, auxiliando no gerenciamento de gastos, planejamento e organização do dinheiro no dia a dia.

## 🎯 Objetivo

Facilitar a vida financeira dos usuários, permitindo um melhor controle sobre receitas e despesas, incentivando hábitos financeiros mais saudáveis.
